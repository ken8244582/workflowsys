# 项目需求说明

> 本文件为流程管理平台的需求管理文档，由项目负责人维护。新增/变更需求请按模板追加，状态变更及时更新。

## 项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 流程管理平台 |
| 项目描述 | 职能流程管理平台，提供流程架构管理、流程清单查询、修订记录跟踪、修订计划管理、端到端流程梳理等功能，配套用户认证与权限系统 |
| 技术栈 | Next.js 16 + React 19 + TypeScript 5 + shadcn/ui + Tailwind CSS 4 + Recharts + Supabase PostgreSQL |
| 维护人 | 肯肯 |

### R004 - 修订计划管理

- **模块**: 职能流程 - 修订计划
- **优先级**: P0
- **状态**: 已完成
- **提出日期**: 2026-06-01
- **完成日期**: 2026-06-08

**需求描述**:
修订计划全生命周期管理：
- 计划列表：分页浏览，支持创建/下发/完成
- 计划详情 `/functional/plan/[id]`：
  - 从流程清单选择添加任务（7列信息展示：流程编码/L4职能流程/版本号/L4所有者/部门/格式/分类）
  - 任务状态流转：待执行 → 进行中 → 已完成
  - 撤回已完成任务：已完成 → 待执行（清除完成时间）
  - 顺延任务：创建下月计划 + 复制任务，原任务标记"已顺延"
  - 导出Excel：Sheet1任务明细 + Sheet2部门完成进度统计
- 修订要求：文本框输入，500字限制
- 表格冻结表头和操作列（不透明背景）
- 截断字段悬浮提示显示完整内容

**验收标准**:
- [x] 计划创建/下发/完成流程正常
- [x] 从流程清单添加任务数据正确
- [x] 任务状态流转（完成/撤回/顺延）正常
- [x] 顺延自动创建下月计划
- [x] 导出Excel含任务明细+部门统计两个Sheet
- [x] 分页组件与流程清单页一致

### R007 - 用户登录与权限

- **模块**: 系统管理
- **优先级**: P1
- **状态**: 已完成
- **提出日期**: 2026-06-04
- **完成日期**: 2026-06-05

**需求描述**:
基于自建 JWT + bcrypt 实现用户登录体系：
- 用户名密码登录（bcrypt验证 → JWT token存入cookie）
- 菜单权限控制（sys_menus + sys_user_menus 动态菜单渲染）
- 用户管理 `/system/users`：用户CRUD + 权限分配 + 启用/禁用
- 菜单管理 `/system/menus`：菜单树CRUD + 排序
- 修改密码功能（首次登录强制修改）
- 超管拥有所有菜单权限

**验收标准**:
- [x] 登录/登出流程正常
- [x] 菜单根据权限动态展示
- [x] 修改密码功能正常
- [x] 时间统一使用北京时间
