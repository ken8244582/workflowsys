## 项目信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 流程管理平台 |
| 项目描述 | L1-L4 职能流程文件清单统计分析平台，提供流程数据统计概览、流程清单查询和修订计划跟踪 |
| 技术栈 | Next.js 16 + React 19 + TypeScript 5 + shadcn/ui + Tailwind CSS 4 + Recharts |
| 维护人 | 肯肯 |

## 需求清单

### R001 - 统计概览页

- **模块**: 统计概览
- **优先级**: P0
- **状态**: 已完成
- **提出日期**: 2026-06-01
- **完成日期**: 2026-06-03

**需求描述**:
首页展示流程管理三段式统计概览：
1. 职能流程工作：4指标卡 + 横向堆叠柱状图 + 双饼图 + 版本分布 + 明细表
2. 端到端流程工作：4指标卡 + 贯通进度图 + 明细表 + 工作说明
3. 流程治理运营工作：4指标卡 + 部门进度图 + 完成情况饼图

**验收标准**:
- [x] 三段式布局完整展示
- [x] 指标卡数据正确
- [x] 图表交互正常（hover 提示、图例筛选）
